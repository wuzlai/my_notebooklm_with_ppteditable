**副标题**: 跨工厂 STO 报表（高复杂度）深度测试

**内容要点**:
- **效率倒挂**: Claude Code 耗时 8 人天，比手写开发（5天）慢 60%。

**视觉建议**: 绘制错误数量趋势折线图，展示“修复-爆炸”曲线；使用断裂的链条图标表示逻辑中断。
